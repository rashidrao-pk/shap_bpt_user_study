E8 – User Preference Study

We use a within-subject, forced-ranking design: for each trial, participants see four explanations side-by-side and order them from 1 (most helpful) to 4 (least helpful).

This yields ordinal data suitable for non-parametric tests (e.g., Friedman and Nemenyi), and lets us compare BPT with baseline methods in terms of perceived intuitiveness.

How this study works

In each trial, you will see one input image and four explanation maps produced by different methods. The underlying CNN classifier (e.g., ResNet) predicts one ImageNet class with some probability.

Your task is to:

  1. Look at the input image and the predicted class.
  2. Ask yourself: “Where should this class appear in the image?”
  3. Inspect the four explanations and decide which one best localizes the object.
  4. Rank the four explanations from 1 (most helpful) to 4 (least helpful).
